# Error Logs - API Documentation

## Overview

The Error Logs feature provides global-admin endpoints to read Countly server log files and clear selected logs.

## Quick Links

- [Error Logs - Logs Read](o-errorlogs.md)
- [Error Logs - Log Read](o-errorlogs-log.md)
- [Error Logs - Log Clear](i-errorlogs.md)

## Configuration & Settings

Error log endpoints read files from the server `log/` directory and discover additional `countly-*.log` files at runtime.
